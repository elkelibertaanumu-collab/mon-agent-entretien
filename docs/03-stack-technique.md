# 03 - Stack technique (recommandation)

## Frontend
- PWA: React + Vite
- UI simple mobile-first
- Enregistrement micro: Web MediaRecorder API

## Backend
- Node.js (Express ou Fastify)
- Endpoints session + analyse

## IA
- STT: transcription audio
- LLM: génération questions + feedback
- TTS (optionnel en V2)

## Données
- PostgreSQL (sessions, scores, utilisateurs)
- Stockage audio objet (S3 compatible)

## Paiement
- Intégration locale simple (mobile money / lien)
- Stripe si disponible selon pays

## Observabilité
- Logs structurés
- Mesures: coût IA/session, latence, taux de complétion
