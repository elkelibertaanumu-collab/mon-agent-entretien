# 02 - Produit MVP

## Expérience utilisateur cible
1. L'utilisateur ouvre la PWA.
2. Il dit: "Je vise un poste de ...".
3. L'agent lance la simulation.
4. L'utilisateur répond à chaque question à voix haute.
5. L'agent donne un feedback final actionnable.

## Fonctionnalités MVP
- Capture audio navigateur
- Speech-to-text
- Génération dynamique de questions
- Scoring simple: clarté, structure, pertinence, confiance
- Rapport de fin de session

## Hors scope MVP
- CV parsing avancé
- marketplace de coachs
- gamification lourde
- multi-langues complexes
